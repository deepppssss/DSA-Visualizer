# DSA Visualizer

An interactive web-based application for visualizing fundamental Data Structures and Algorithms through real-time animations.

## Features

### Sorting Algorithms

* Bubble Sort
* Merge Sort
* Quick Sort

### Searching Algorithms

* Binary Search

### Graph Algorithms

* Breadth First Search (BFS)
* Depth First Search (DFS)
* Dijkstra's Algorithm

## Functionality

* Real-time algorithm visualization
* Adjustable animation speed
* Dynamic input generation
* Step-by-step execution
* Responsive user interface

## Tech Stack

* HTML
* CSS
* JavaScript

## Deployment

Deployed using Vercel
